# 芯图相册使用指南

欢迎使用芯图相册！本指南将帮助您快速上手，分别介绍PC端（Windows）和移动端（Android）的使用方法。

## 📥 安装应用

### 下载地址

请访问官方网站 **www.xintuxiangce.top** 下载最新版本。

### 安装说明

#### PC端（Windows）

PC端提供两种版本供您选择：

**1. 安装包**
- 需要安装到系统
- 安装后会在开始菜单和桌面创建快捷方式
- 适合长期使用，方便快速启动
- 安装步骤：
  1. 下载安装包文件(www.xintuxiangce.top)
  2. 双击安装包进行安装
  3. 按照安装向导完成安装
  4. 安装完成后，在开始菜单或桌面找到"芯图相册"图标启动应用

**2. 便携包**
- 无需安装，解压即用
- 可以放在任意位置（如U盘、移动硬盘）
- 适合临时使用或不想在系统安装软件的用户
- 使用步骤：
  1. 下载便携包压缩文件
  2. 解压到任意目录（建议解压到非系统盘）
  3. 进入解压后的文件夹，双击运行"XinTuAlbum 1.0.0.exe"即可使用
  4. 删除文件夹即可完全卸载，不留任何痕迹

> **选择建议**：
> - 如果您是长期使用，建议选择**安装包**，更方便快捷
> - 如果您想随时携带使用或不想在系统安装，建议选择**便携包**

![PC版本初始启动](./images/guide/1_PC-start.webp)

#### 移动端（Android）

1. 下载Android安装包（.apk格式）
2. 在手机上打开下载的安装包
3. **重要提示**：由于目前受软件著作权申请流程影响，应用尚未上架到应用商店，因此安装时系统会显示"安全风险提示"
4. 这是正常现象，请点击"继续安装"或"允许安装此应用"
5. 安装完成后，在应用列表中找到"芯图相册"图标启动应用

> **说明**：应用正在申请软件著作权，完成后将上架到各大应用商店，届时可直接从应用商店下载，无需担心安全提示。

![Android版本初始启动](./images/guide/2_Android-start.webp)

## ⚙️ 初始设置

### 设置照片目录（可选）

安装并运行应用后，建议先进入设置页面配置照片扫描目录。

#### PC端（Windows）

1. 启动应用后，点击右上角的**设置**按钮（⚙️图标）
![PC版进入设置](./images/guide/3_PC-enter-settings.webp)
2. 在设置页面找到"照片目录"设置项
![PC版照片目录](./images/guide/4_PC-settings-directory.webp)
3. **默认设置**：应用会自动初始化为用户文档的图片目录（如：`C:\Users\用户名\Pictures`）
4. 您可以根据需要修改为其他目录：
   - 点击"选择目录"按钮
   - 在文件浏览器中选择您想要扫描的照片文件夹
   - 可以添加多个目录
5. 设置完成后，点击"添加"
![PC版本目录添加](./images/guide/5_PC-directory-save.webp)

#### 移动端（Android）

1. 启动应用后，点击底部导航栏的**设置**标签
![Android版本进入设置](./images/guide/6_Android-enter-settings.webp)
2. 在设置页面找到"照片目录"设置项
![Android版本目录设置](./images/guide/7_android-settings-directory.jpg)
3. **默认设置**：移动端初始化为空，表示不限制扫描范围
4. **如果不设置目录**：点击扫描按钮时，应用会对手机的所有照片进行扫描
5. **如果设置目录**：应用只会扫描您指定的目录中的照片
6. 设置目录的方法：
   - 点击"添加目录"按钮，手动选择目录
   - 或使用**快捷目录按钮**快速添加常用目录：
     - **💬 微信目录**：一键添加微信图片目录（适合只想整理微信图片的用户）
     - **💬 QQ目录**：一键添加QQ图片目录
     - **📷 相册目录**：一键添加相机相册目录
     - **📸 截图目录**：一键添加截图目录
7. 设置完成后，应用会自动保存

> **使用建议**：
> - 如果您想整理所有照片，可以不设置目录，直接使用默认设置
> - 如果您只想整理特定应用的图片（如微信），可以点击"微信目录"快捷按钮，快速设置微信图片目录

## 🔍 开始扫描

![PC版本扫描](./images/guide/8_PC-start-scan.png)

### 扫描步骤

1. 在首页点击右下角的**扫描按钮**（🔄图标）
2. 应用会开始扫描您设置的照片目录（或全部照片）
3. 扫描过程中会显示进度信息
4. 扫描完成后，照片会自动分类显示

### 非会员限制说明

**重要提示**：非会员用户每次扫描只会对**最近的100张照片**进行分类处理。

- 如果您有大量照片需要整理，建议先开通会员（详见下方"成为会员"章节）
- 会员用户可以无限制扫描和分类所有照片

扫描时，如果检测到您是非会员，应用会在扫描按钮附近显示提示信息："为相册智能分类100张，在设置页面开通终身会员后，无限制"

## 💎 成为会员

### 会员权益

开通会员后，您将享受以下权益：

- ✅ **无限制扫描**：可以扫描和分类所有照片，不受100张限制
- ✅ **无限制分类**：所有照片都能享受AI智能分类
- ✅ **完整功能**：解锁所有高级功能
- ✅ **终身会员**：一次开通，终身使用

### 如何开通会员

#### 方法一：通过应用内开通（推荐）

1. 打开应用，进入**设置**页面
2. 找到"会员服务"区域
3. 如果显示"未激活"，会显示一个二维码
4. 点击二维码，应用会：
   - 自动保存二维码到相册
   - 尝试打开微信扫一扫功能
5. 使用微信扫描二维码，关注"芯图相册"公众号
6. 在公众号中按照提示开通会员
7. 开通成功后，返回应用，会员状态会自动更新为"已激活"

![成为会员](./images/guide/9_membership.webp)

#### 方法二：直接关注公众号

1. 打开微信，搜索并关注"芯图相册"公众号
2. 在公众号菜单中找到"开通会员"
3. 按照提示完成会员开通流程
4. 开通成功后，在应用的设置页面刷新会员状态

### 会员状态查询

- 在设置页面的"会员状态"区域可以查看当前会员状态
- 会员用户会显示"已激活"和"使用额度"信息
- 非会员用户会显示"未激活"和开通二维码

## ❓ 常见问题

### Q: 为什么移动端安装时显示安全风险提示？

A: 由于应用正在申请软件著作权，尚未上架到应用商店，因此系统会显示安全提示。这是正常现象，应用本身是安全的，可以放心安装使用。

### Q: 非会员只能分类100张照片吗？

A: 是的，非会员每次扫描只处理最近的100张照片。开通会员后可以无限制扫描和分类所有照片。

### Q: 如何只整理微信的图片？

A: 在设置页面的"照片目录"区域，点击"💬 微信目录"快捷按钮，应用会自动检测并添加微信图片目录。

### Q: 扫描需要联网吗？

A: 应用支持两种模式：

**联网模式（推荐）**：
- 扫描时需要联网
- 使用云端大模型的照片理解能力对图片进行分类
- 分类效果更准确，识别更智能
- 所有处理结果保存在本地，原始照片服务器侧不会存储

**离线模式**：
- 扫描时无需联网
- 使用本地小模型进行图像识别后分类
- 分类效果不如大模型理想，但可以满足基本需求
- 完全本地处理，保护隐私

> **建议**：如果网络条件允许，建议使用联网模式以获得更好的分类效果。

### Q: 会员是永久的吗？

A: 是的，开通会员后是终身会员，一次付费，永久使用。
