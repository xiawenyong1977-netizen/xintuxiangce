常见问题

解答您的疑惑

芯图相册真的完全免费吗？

+

是的，芯图相册完全免费使用，无任何隐藏收费，也没有内置广告。我们希望为每个用户提供纯粹、高效的照片管理体验。

我的照片真的安全吗？会不会被上传？

+

绝对安全！芯图相册的所有AI处理都在您的本地电脑进行，不需要联网，不会上传任何照片到云端。您的照片始终在您的设备上，我们无法访问，也不会收集任何数据。这是我们设计的核心原则。

分类准确率有多高？

+

经过大量测试，芯图相册的分类准确率稳定在90%以上。对于少数分类错误的照片，您可以手动调整。我们也在持续优化算法，未来版本的准确率会更高。

支持哪些操作系统？

+

目前芯图相册支持 Windows 10 及以上版本。Mac 和 Linux 版本正在开发中，敬请期待。

可以处理多少张照片？

+

理论上没有数量限制。我们测试过处理近2000张、总大小近9GB的照片，都能顺利完成。处理速度取决于您的电脑配置和照片数量。

删除的照片可以恢复吗？

+

芯图相册提供了"暂存箱"功能，在最终删除前可以进行二次确认。一旦确认删除，照片会进入系统回收站，您可以从回收站恢复。为了安全起见，建议在删除前备份重要照片。

对电脑配置有什么要求？

+

建议配置：Windows 10/11，4GB以上内存，500MB可用磁盘空间。对于大批量照片处理，建议8GB以上内存以获得更好的体验。
